# Changelog

All notable changes to this repository will be documented in this file.

This project follows the principles of reproducible research. Each version records significant updates to the framework, experimental methodology, benchmark implementation, and documentation.

The format is based on **Keep a Changelog**, and the project follows **Semantic Versioning (MAJOR.MINOR.PATCH)**.

## [1.0.0] - 2026-07-08

- Initial release of the **SecureSmartContractBenchmark** research framework.
- Modular smart contract architecture for secure blockchain applications.

- Standardized benchmark evaluation metrics.
- Reproducible experimental workflow.
- Hardhat project configuration.

- Smart-grid transaction dataset integration.

## Versioning Policy

- **MAJOR** versions introduce significant research framework updates or architectural changes.
- **MINOR** versions add validated functionality, benchmark modules, or experimental capabilities.
- **PATCH** versions resolve implementation issues, improve documentation, or correct reproducibility-related problems without changing the research methodology.
